FN Amplifications

My first ever Slimefun addon, it features a lof of things from power generators, machineries, block breakers, staffs and many more! It was first released back in november 2021 as an personal addon for a server called BarangayCraft and I further developed it to become an official community addon for slimefun. It was rewritten this february and march to improve code maintainabilty and readability. I have added a lot of things since then and will continute to further develop it. Boomer has also covered the addon when it was released last november.



Since then a lot has changed and updates keep coming that gets featured in boomers weekly slimefun updates. FN Amplifications Github Repository.

SfChunkInfo

My Second addon for slimefun which can scan the amount of slimefun items and blocks inside a 16x16 chunk. It can also scan others chunk and can respect protected land claims



Again another handy addon for big labs and setups. SfChunkInfo Github Repository
